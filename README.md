# dnsix

An IPv6-only DNS64 forwarder written in Rust. It relays every query to a
configured upstream resolver and synthesizes AAAA records ([RFC 6147]) for
IPv4-only names, so IPv6-only clients reach IPv4-only hosts via a companion
NAT64 translator. Beyond the well-known NAT64 fallback, it can point selected
CDN hostnames at their native IPv6 ranges (no translator hop).

> **This project was generated by AI.** While the AI was closely steered, do not deploy onto important networks.

## Motivation

I wanted DNS64 which does CDN tricks in Rust with proper observability (a Prometheus `/metrics` endpoint for Zabbix) for deployment in my IPv6-only homelab.

## Credits

## Usage

```sh
cargo run -- --config config.toml
```

Copy [`config.example.toml`](config.example.toml) to get started; only
`upstreams` is required.
